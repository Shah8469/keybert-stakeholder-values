# KeyBERT Stakeholder Values

This project develops an unsupervised NLP framework using KeyBERT to identify stakeholder-value themes from transportation resilience literature.

## Objectives

- Extract stakeholder-value keyphrases automatically
- Analyze dominant themes in literature
- Evaluate extraction relevance and robustness

## Dataset

Curated corpus of 20+ academic papers covering:

- transportation resilience
- disaster recovery
- infrastructure systems
- climate adaptation

Fields used:

- title
- keywords
- full text

## Methodology

1. Data collection  
2. Text preprocessing  
3. KeyBERT keyphrase extraction  
4. Semantic similarity ranking  
5. Stakeholder-value mapping  
6. Validation using Precision@10  
7. Robustness checks

## Main Results

Top themes identified:

- Sustainability
- Adaptability
- Accessibility
- Equity
